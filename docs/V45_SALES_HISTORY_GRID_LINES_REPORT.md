# V45 Sales History Grid Lines Report

## Nội dung đã chỉnh

- Thêm đường dóng cột dọc mờ cho danh sách lịch sử đơn bán.
- Giữ giao diện nhẹ, không dùng viền đậm.
- Thêm sọc nền xen kẽ rất nhạt giữa các dòng để dễ rà soát đơn.
- Căn giữa cột ngày bán.
- Căn phải cột giá trị.
- Căn giữa cột nguồn và thao tác.
- Giữ header bảng hiện có và làm rõ đường phân cách header/body.

## File đã sửa

- `public/style.css`

## Mục tiêu

Danh sách đơn bán nhìn giống bảng ERP hơn, dễ dóng mắt theo từng cột:

- Mã đơn
- Khách hàng
- Ngày bán
- Giá trị
- Nguồn
- Thao tác

## Ghi chú

Không thay đổi nghiệp vụ, API hay dữ liệu. Chỉ chỉnh CSS hiển thị.
