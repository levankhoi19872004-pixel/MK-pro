IMPORT CENTER V43

Import dữ liệu Excel

Quy trình chuẩn: tải đúng mẫu → điền vào sheet Import → xem trước & kiểm tra lỗi → tick dòng hợp lệ → xác nhận ghi vào hệ thống.

01Tải mẫu Excel

Mẫu có sẵn cột chuẩn và dữ liệu ví dụ.

02Điền dữ liệu

Điền vào sheet Import, không đổi tên cột.

03Xem trước

Hệ thống kiểm tra lỗi trước khi ghi dữ liệu.

04Xác nhận

Chỉ các dòng được tick hợp lệ mới được import.

Thực hiện import

Chọn nghiệp vụ, tải mẫu hoặc chọn file Excel để preview.

XL
Mẫu import sản phẩm

Cột bắt buộc: mã sản phẩm, tên sản phẩm. Có thể nhập thêm đơn vị, barcode, nhóm hàng, giá nhập, giá bán.

Kho mẫu Excel

Bấm tải nhanh theo từng nghiệp vụ.

Nguyên tắc import an toàn: Không ghi thẳng vào tồn kho, công nợ, quỹ tiền nếu chưa qua preview. Không đổi tên cột trong Excel. Sản phẩm và khách hàng nên import trước các chứng từ phát sinh.
