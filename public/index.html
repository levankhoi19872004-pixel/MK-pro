Import dữ liệu Excel

Chọn file rồi bấm Import ngay. Hệ thống tự kiểm tra, tự cắt hàng vượt tồn và báo cáo kết quả sau import.

Mẫu Excel chuẩn:

Bấm Tải mẫu Excel theo đúng loại import đang chọn. Sau đó chọn file và bấm Import ngay.

Nếu file Excel có tên cột riêng, dùng khu vực Tự tạo mẫu import để map cột Excel về trường dữ liệu.

Sản phẩm/khách hàng nên import trước, sau đó mới import tồn kho, phiếu nhập, đơn bán, công nợ và quỹ tiền.

Tự tạo mẫu import

Tự đặt tên cột Excel, chọn trường database tương ứng rồi lưu thành mẫu dùng lại.

Hệ thống

Backup, kiểm tra dữ liệu và reset phần mềm khi cần làm sạch dữ liệu.

Mongo: Đang kiểm tra Reset: Đang kiểm tra Nguồn dữ liệu: MongoDB

1. Backup dữ liệu

Nên backup trước khi reset. File backup sẽ lưu trong thư mục backups trên server.

2. Reset dữ liệu phần mềm

Reset sẽ tự tạo backup trước, sau đó xóa dữ liệu theo phạm vi bên dưới.

Chức năng này chỉ chạy khi server bật ALLOW_SYSTEM_RESET=true.
