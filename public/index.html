Import dữ liệu Excel

Xem trước dữ liệu, kiểm tra lỗi, tick chọn dòng hợp lệ rồi mới ghi vào hệ thống.

Mẫu Excel chuẩn:

Bấm Tải mẫu Excel theo đúng loại import đang chọn. Mỗi file có 3 sheet: HuongDan, DuLieuMau và Import.

Nếu file Excel của bạn có tên cột riêng, dùng khu vực Tự tạo mẫu import bên dưới để map cột Excel về trường dữ liệu.

Sản phẩm/khách hàng nên import trước, sau đó mới import tồn kho, phiếu nhập, đơn bán, công nợ và quỹ tiền.

Tự tạo mẫu import

Tự đặt tên cột Excel, chọn trường database tương ứng rồi lưu thành mẫu dùng lại.
