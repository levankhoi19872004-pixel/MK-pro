Import dữ liệu Excel

Chọn file rồi bấm Import ngay. Hệ thống tự kiểm tra, tự cắt hàng vượt tồn và báo cáo kết quả sau import.

Mẫu Excel chuẩn:

Bấm Tải mẫu Excel theo đúng loại import đang chọn. Sau đó chọn file và bấm Import ngay.

Nếu file Excel có tên cột riêng, dùng khu vực Tự tạo mẫu import để map cột Excel về trường dữ liệu.

Sản phẩm/khách hàng nên import trước, sau đó mới import tồn kho, phiếu nhập, đơn bán, công nợ và quỹ tiền.

Cập nhật an toàn chỉ áp dụng cho Sản phẩm, Khách hàng và Tài khoản: ô trống/cột thiếu giữ nguyên dữ liệu cũ; chỉ field có giá trị mới và khác dữ liệu Mongo mới được cập nhật.
