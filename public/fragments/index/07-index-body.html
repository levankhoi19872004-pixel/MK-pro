Mongo: Đang kiểm tra Reset: Đang kiểm tra Nguồn dữ liệu: MongoDB

1. Backup dữ liệu

Nên backup trước khi reset. File backup sẽ lưu trong thư mục backups trên server.

2. Reset dữ liệu phần mềm

Reset sẽ tự tạo backup trước, sau đó xóa dữ liệu theo phạm vi bên dưới.

Chức năng này chỉ chạy khi server bật ALLOW_SYSTEM_RESET=true.
